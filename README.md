## Concepts learned
- **Backend principles**
  - API Rest;
  - SQL databases;
  - Basics of Clean Archtecture;
  - Error handling;
  - Deploing with Docker;
  - Editing DB with ORM;
  - Schemas, Migrations, request validation;
  - DotEnv variables.
- **Frontend principles**
  - React components;
  - React hooks, useState, useEffect, useQuery;
  - Api implementation;
  - TailwindCSS custom configs.
- Vercel Deployment.
